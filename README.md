# Kid AI Tutorial

A 10-mission, no-install tutorial that teaches a kid to go from "what is a terminal" to "I can call Claude from Python" in a single afternoon.

## Open it

The tutorial is a static HTML page — no server, no install. Just open one of the files in a modern browser (Firefox or Chrome both work):

- `index.html` — English version
- `index-bg.html` — Bulgarian version (Български)

Progress and the optional Anthropic API key are saved in your browser's `localStorage`, so you can close the tab and come back later without losing where you were. (Private/incognito mode skips this — the page will warn you.)

## What's inside

Each mission has a short reading section, copy-ready code blocks, a hands-on challenge, and a small quiz at the end. Missions unlock one at a time so the kid finishes the previous one before opening the next.

Mission 1 also includes an "open commands reference" panel that explains every terminal command used in the tutorial with two worked examples each, plus a step-by-step guide for creating files three different ways (`touch`, `echo >`, and Python's `open()`).

From Mission 4 onward (once an Anthropic API key is saved), the kid can chat with Claude directly inside the page — both through the inline "Try it yourself" prompt in Mission 3 and the "Ask Claude" bar pinned to the bottom of the page.

## Found a bug? Want to give feedback?

There's a floating 💬 button in the bottom-right corner of the page. Click it, describe what didn't work or what you'd like improved, and hit Send — it pre-fills a GitHub issue in a new tab and you can submit it with one click.

You can also open issues directly at <https://github.com/derwin4o/kid-ai-tutorial/issues>.
